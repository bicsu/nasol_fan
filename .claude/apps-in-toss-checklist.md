## 지금 바로 할 수 있는 일 (사업자 수령 전)

1. **appName 확정** — 등록 후 변경 불가이므로 신중히 (예: `nasolfans`)
2. **앱 로고 최종본** — 1024x1024 PNG (현재 icon.png 있음, 확인 필요)
3. **고객문의 이메일** — 사업용 이메일 생성
4. **개인정보처리방침 초안** — 수집항목: 닉네임, 토스 유저키
5. **서비스 이용약관 초안** — 커뮤니티 운영 규칙 포함
6. **푸시 알림 템플릿 문구 설계** — 방영 알림, 투표 결과 등
7. **백엔드 서버 구조 설계** — mTLS 적용할 서버 아키텍처
8. **앱 번들 용량 체크** — `npx expo export` 후 100MB 이하 확인
9. **약관 동의 UI 구현** — 토스 로그인 시 필요한 약관 동의 화면
10. **만 19세 이상 안내** — 앱 진입 시 또는 약관에 포함

## 타임라인 예상

```
사업자 수령 (D+0)
  → 개발자센터 가입 + 앱 등록 (D+1)
  → 앱 등록 검수 (D+3)
  → mTLS 인증서 발급 (D+3)
  → 토스 로그인 연동 개발 (D+3 ~ D+14)
  → 푸시 알림 연동 (D+14 ~ D+21)
  → 앱 번들 업로드 + 출시 검수 (D+21 ~ D+24)
  → 출시 (D+24~)
```

비게임 앱 기준 약 2~3개월 소요 (공식 안내), 단순 커뮤니티는 더 빠를 수 있음
