## 1. 토스 콘솔 설정

- [ ] **토스 로그인 Client ID / Secret 확인**
  - 앱인토스 콘솔 → 나솔팬즈 → 좌측 **"토스 로그인"** 메뉴
  - `TOSS_CLIENT_ID`, `TOSS_CLIENT_SECRET` 값 복사

- [ ] **토스 푸시 알림 템플릿 3종 등록**
  - 앱인토스 콘솔 → **"푸시알림"** 메뉴
  - 방영 30분 전 알림: `"나솔 {{episode}}회 곧 시작! 채팅방 입장하기"`
  - 투표 결과: `"예측 적중! +{{points}}p 획득"`
  - 인기글 알림: `"HOT 게시글이 올라왔어요"`
  - 발급된 `templateSetCode` 3개 메모

- [ ] **mTLS 인증서 확인**
  - 앱인토스 콘솔 → **"mTLS 인증서"** 메뉴
  - `server/certs/` 에 있는 파일이 맞는지 확인

## 2. Vercel 환경변수 추가

> Vercel 대시보드 → 프로젝트 → Settings → Environment Variables

- [ ] `TOSS_CLIENT_ID` — 토스 콘솔에서 복사
- [ ] `TOSS_CLIENT_SECRET` — 토스 콘솔에서 복사
- [ ] `TOSS_CERT_BASE64` — 터미널: `base64 -i server/certs/nasol_fans_public.crt | pbcopy`
- [ ] `TOSS_KEY_BASE64` — 터미널: `base64 -i server/certs/nasol_fans_private.key | pbcopy`
- [ ] `TOSS_CA_BASE64` — CA 파일 있으면: `base64 -i server/certs/toss-ca.crt | pbcopy`
- [ ] `SUPABASE_SERVICE_KEY` — `.env` 파일의 `supabase_secret_key` 값
- [ ] `CRON_SECRET` — 아무 랜덤 문자열 (예: `openssl rand -hex 32`)
- [ ] `FRONTEND_URL` — Vercel 배포 URL (예: `https://nasolfans.vercel.app`)
- [ ] `TOSS_AUTH_BASE_URL` — `https://apps-in-toss-api.toss.im`
