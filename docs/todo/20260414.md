## 2. Vercel 환경변수 추가

> Vercel 대시보드 → 프로젝트 → Settings → Environment Variables

- [ ] `TOSS_CLIENT_ID` — 토스 콘솔에서 복사
- [ ] `TOSS_CLIENT_SECRET` — 토스 콘솔에서 복사
- [ ] `TOSS_CERT_BASE64` — 인증서 배치 후: `base64 -i server/certs/nasol_fans_public.crt | pbcopy`
- [ ] `TOSS_KEY_BASE64` — 인증서 배치 후: `base64 -i server/certs/nasol_fans_private.key | pbcopy`
- [ ] `TOSS_CA_BASE64` — CA 파일 있으면: `base64 -i server/certs/toss-ca.crt | pbcopy`
- [ ] `SUPABASE_SERVICE_KEY` — `.env` 파일의 `SUPBASE_SERVICE_ROLE_KEY` 값 (`eyJhbGci...` 로 시작하는 값)
- [ ] `CRON_SECRET` — 터미널: `openssl rand -hex 32`
- [ ] `FRONTEND_URL` — Vercel 배포 URL (예: `https://nasolfans.vercel.app`)
- [ ] `TOSS_AUTH_BASE_URL` — `https://apps-in-toss-api.toss.im`
