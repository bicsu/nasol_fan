  1. 서비스 개요                                                                                                                                         
                                                            
  나솔팬즈의 채팅 기능은 특정 이용자 간 1:1 메시지 교환이 아닌, TV 프로그램 "나는 솔로" 방영 시간에 맞춰 자동 개설되는 공개 실시간 채팅방입니다.         
  
  - 랜덤 매칭, 근처 사용자 탐색, 이성 추천 등 만남·소개팅 기능 없음                                                                                      
  - 방영 시작 시 자동 개설, 방영 종료 후 1시간 뒤 읽기 전용 아카이브로 전환
  - 만 19세 이상 토스 로그인 이용자만 참여 가능                                                                                                          
                                                                                                                                                         
  ──────────────────────────────────────                                                                                                                 
                                                                                                                                                         
  2. 이용자 보호 기능                                       

  [신고 기능]                                                                                                                                            
  - 채팅 메시지를 길게 누르면 신고 기능에 즉시 접근 가능
  - 신고 사유: 스팸/도배, 욕설/비방/혐오, 불법 정보/사기, 기타                                                                                           
  - 신고 내역은 데이터베이스에 저장되며, 운영자가 주기적으로 검토                                                                                        
  - 신고 상태: 접수(pending) → 검토완료(reviewed) / 기각(dismissed) 으로 관리                                                                            
                                                                                                                                                         
  [차단 기능]                                                                                                                                            
  - 채팅 메시지를 길게 누르면 차단 기능에 즉시 접근 가능                                                                                                 
  - 차단 즉시 해당 이용자의 메시지가 클라이언트에서 숨김 처리
  - 차단 목록은 데이터베이스에 저장되며 언제든 해제 가능                                                                                                 
                                                                                                                                                         
  [계정 제재 정책]                                                                                                                                       
  - 동일 이용자에 대한 신고가 3회 이상 누적될 경우 운영자 검토 후 계정 이용 제한 또는 정지 조치                                                          
  - 심각한 위반(불법 정보 유통, 사기 등)의 경우 즉시 계정 정지                                                                                           
                                                                                                                                                         
  [신고 대응 프로세스]                                                                                                                                   
  - 신고 접수 → 운영자 검토(영업일 기준 48시간 이내) → 위반 확인 시 콘텐츠 삭제 및 계정 조치 → 결과 보관                                                 
  - 운영 이메일: nasolfans@gmail.com                                                                                                                     
                                                                                                                                                         
  ──────────────────────────────────────                                                                                                                 
                                                                                                                                                         
  3. 불법 행위 방지                                         

  이용약관(제7조, 제15조의2)을 통해 아래 행위를 명시적으로 금지하며, 신고 시스템을 통해 적극 관리합니다.                                                 
  
  - 성매매 또는 조건만남을 유도하는 광고·메시지                                                                                                          
  - 금전 요구 또는 투자 권유를 통한 사기 행위 (로맨스 스캠 포함)
  - 불법 광고 및 스팸 메시지                                                                                                                             
  - 불법 정보 유통                                                                                                                                       
  
  위반 콘텐츠 발견 또는 신고 접수 시 즉시 삭제 조치하며, 필요 시 수사기관에 협조합니다.

  4. 개인정보 및 메시지 데이터 보호

  - 최소 수집: 닉네임, 토스 유저키만 수집 (토스 로그인 연동 후)                                                                                          
  - 암호화: 데이터베이스 저장 데이터는 TLS 암호화 적용
  - 탈퇴 시 파기: 탈퇴 시 닉네임, 토스 유저키 즉시 파기, 채팅·게시물은 작성자 정보 익명 처리("알 수 없음")                                               
  - 로그 보관: 이용자 보호 및 법적 대응 목적으로 채팅 로그 90일 보관 후 삭제
