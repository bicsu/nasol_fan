## 참고 링크
- 앱인토스 채팅 가이드라인: `docs/채팅주의사항.txt`
- Supabase 대시보드: https://supabase.com/dashboard/project/fbvgnxpnznfpnprypyyl
- 신고 내역 조회 SQL:
  ```sql
  SELECT r.*, 
    u1.nickname AS reporter,
    u2.nickname AS reported_user
  FROM reports r
  JOIN users u1 ON r.reporter_id = u1.id
  JOIN users u2 ON r.reported_user_id = u2.id
  WHERE r.status = 'pending'
  ORDER BY r.created_at DESC;
  ```
